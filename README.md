## Setup

1. **Clone the repo**

```
$ git clone https://github.com/githubityu/flutter_jdshop_mvvm.git
$ cd flutter_jdshop_mvvm
```

2. **Running:**

```
$ flutter run
```




## 简介

这是一个用Flutter写的简易商城APP
